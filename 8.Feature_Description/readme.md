### 摘要：   
我认为三维点的核心在于点坐标的描述子，如果能准确快速的对点进行描述，就可以实现后面很多核心的任务。    
实现了FPFH描述子，测试的时候对modelnet中的模型做一个旋转，取旋转前后的两个对应点坐标P_anchor和P_pos，分别提取描述子，再取一个其他的点提取描述子作为P_neg，对这三个点分别提取描述子，计算两两之间的距离。
